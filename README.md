# trainer でできること

Arduino を使ったプログラミング学習用の教材です．段階を踏んで様々なことを学ぶことができるよう開発を進めています．

## 本リポジトリで管理されているもの

- 専用の開発ボード
  - `./boards/`
- 各種サンプルコード
  - `./samples/`
- 各種ドキュメント（Web ページとして公開）
  - `./docs/`
- 関連リンク

## ハードウェア

実習用に，秋月電子で購入できる RP2040 搭載のマイコンボード「AE-RP2040」を使用するハードウェア「trainer-pico」を用意しています．
