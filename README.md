# trainer でできること

Arduino を使ったプログラミング学習用の教材です．段階を踏んで様々なことを学ぶことができるよう開発を進めています．

## 本リポジトリで管理されているもの

- 専用の開発ボード
  - `./boards/`
- 各種サンプルコード
  - `./samples/`
- 各種ドキュメント（Web ページとして公開）
  - `./docs/`
- 関連リンク

## trainer シリーズのバリエーション

以下の 2 種類の開発ボードを提供しています．

- trainer-nano
  - `./boards/trainer-nano/`
  - Arduino Nano を使用します
- trainer-pico
  - `./boards/trainer-pico/`
  - Raspberry Pi Pico を使用します
